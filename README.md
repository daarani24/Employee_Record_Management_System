# Employee Record Management System

A **console-based Python application** developed using **Object-Oriented Programming (OOP)** concepts to efficiently manage employee records.

## Features

* Add Employee
* View All Employees
* Search Employee
* Update Employee Details
* Delete Employee
* Highest Salary Employee Report
* Experienced Employee Report
* Save Records to File

## OOP Concepts Used

* Encapsulation
* Inheritance
* Abstraction
* Polymorphism
* Classes & Objects
* Constructors
* Getters & Setters

## Project Structure

```text
Employee_Record_Management_System/
│
├── employee.py
├── employee_manager.py
├── report.py
├── file_handler.py
├── main.py
└── data.txt
```

## Technologies Used

* Python 3
* Visual Studio Code

python main.py
